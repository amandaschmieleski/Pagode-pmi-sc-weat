# Pagode-pmi-sc-weat

Este repositório reúne os arquivos utilizados em um estudo sobre associações semânticas em letras de pagode, utilizando métricas como **PMI** e **SC-WEAT**.

O objetivo do projeto é analisar viés de gênero em músicas brasileiras.

## Estrutura do repositório

- `Códigos/`: scripts utilizados no processamento e nas análises.
- `Dados/`: arquivos de entrada e resultados gerados.
- `Listas de alvos e atributos/`: listas de palavras usadas nas categorias de análise.

## Metodologia geral

O fluxo do projeto envolve:

1. preparação e limpeza dos dados;
2. organização das listas de alvos e atributos;
3. cálculo de associações semânticas por PMI;
4. aplicação do SC-WEAT;
5. análise dos resultados obtidos.
